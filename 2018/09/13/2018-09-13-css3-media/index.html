浅谈css3media媒体查询

在没有css3的时候其实是有media媒体查询的

<link rel="stylesheet" type="text/css" href="" media="screen">
代表浏览器渲染的css
<link rel="stylesheet" type="text/css" href="" media="print">
代表打印机渲染的css

css3媒体查询

语法为@media mediatype and|not|only (meida feature){}

demo:

当然，该媒体查询能使用‘only来解决ie8或更老旧版本浏览器的兼容问题

原理:老旧浏览器读取上面媒体查询的内容得到的结果为media="only"，然而老旧浏览器的媒体查询没有only这个属性，所以默认是不会渲染该媒体的样式

媒体功能

值 描述
width 视口宽度
height 视口高度
device-width 渲染表面的宽度，即设备屏幕的宽度
device-height 渲染表面的高度，即设备屏幕的高度
orientation 检查设备处于横向还是纵向
aspect-ratio 基于视口宽度和高度的宽高比
device-aspect-ratio 渲染表面的宽度和高度的宽高比
color 每种颜色的位数（bits），如:min-color：16
resolution 检测屏幕或打印机的分辨率（dpi）

以上属性均可添加前缀max-或min-

视口：viewport

该内容主要针对移动端

移动端的视口有三个：布局视口（layout viewport）、可视视口（visual viewport）和理想视口（ideal viewport）

布局视口：页面渲染后的分辨率，可大于可视视口

可视视口：设备的分辨率

理想视口：布局视口在一个设备上的最佳尺寸

一般实现理想窗口的方法
<meta name="viewport" content="width=device-width">
<meta name="viewport" content="width=device-width,maximum-scale=1.0,minimum-scale=1.0,user-scalable=no">
