上一个博文介绍了jekyll的安装和项目部署，下面就略讲一下jekyll博客文章的语法

一、标题

书写方式：“方式”+空格+内容
标题的写法有五种：

1.## 大标题

2.### 小标题

3.#### 超小标题

4.=== 超大标题

5.- - - 标题

效果如下：

大标题

小标题

超小标题

超大标题

标题

二、行内代码

语法：两个 ` 之间加入代码

例子：

echo json_encode($result);

三、引用

语法：> 、 > > + 内容

例子：

This is a example

This is a example

like this

四、列表

1.无序列表

语法：*、-、+加内容

例子：

  • C
  • Java
  • C#
  • PHP
  • Ruby
  • python
  • Lua
  • Javascript
  • shell

2.有序列表

语法：数字+内容

例子：

  1. 高等代数
  2. 数学分析
  3. 离散数学
  4. 数据结构

五、分割线

语法：***、---、\n +> + \n

例子：

* * *

***

*****

- - -

---

六、超链接

语法：[title](url)

例子：
百度

七、强调（加粗、斜体）

语法：*内容*、**内容**、_内容_、__内容__

例子：

内容1

*内容1*

内容2

**内容2**

内容3

_内容3_

内容4

__内容4__

八、图片引用

语法：![alt](图片路径)

九、简单链接与邮箱

语法：<内容>

例子：
https://www.baidu.com
951387826@qq.com

十、代码块

语法：四个空格或一个Tab

例子：
echo $name;
