这篇博文主要是记下三个常用数据库的部分语法不同点（可能不完整，待更新）

一、时间条件

1.mysql

直接两个单引号即可

例：select * from test where time > '2017-10-23'

2.sql server

与mysql一样以加单引号即可

例：select * from test where time > '2017-10-23'

3.oracle

需要使用转换函数to_date()

例：select * from test where time > to_date('20171023','yyyyMMdd')

二、数据分页

1.mysql

使用limit

例：select * from test limit 0,25

2.sql server

利用rownumber

例：select * from test where rownumber > 0

3.oracle

利用rownum

例：select * from test where rownum > 0

三、自定义排序

1.mysql

利用field函数

例：select * from test order by field(字段名,'值','值','值','值')

2.sql server

暂时好像没有专门的函数解决这个问题，建议用case解决

3.oracle

利用decode函数

例：select * from test order by decode(字段名,'值1',1,'值2',2,'值3',3,'值4',4)
